# Règles de calcul en Python
## Opérateurs

```
* : pour la multiplication
** : pour les puissances
/ : pour la division (toujours float)
// : pour la division (toujours integer sauf si float utilisé dans le calcul)
```

## Entier, flottants, notation scientifique et bases

```python
print(2)
2
print(2.)
2.0
print(2e10)
20000000000.0
print(0x111)
237
print(0b111)
7
print(0o111)
73
```
